# Backend

## Install

Windows: You have to run next command in WSL2 terminal from root project.
```
./vendor/bin/sail up
```

# Frontend

## Install
```
npm install
npm run dev
```

## Use translations (i18n)
```
this.$t('title')
```

## Use icons

```
<v-icon icon="mdi-domain" />
<font-awesome-icon :icon="['fas', 'home']" />
```
